## Personal Site
This is the repo for my personal website. The live site can be reached [here](https://www.maxwellmcneal.com).

## Tech stack
This website is built using [Astro](https://astro.build/).

The theme of this blog has been built using the Astro Starter Kit: Blog template, use the shell command below to generate a new project with the same template.

```sh
npm create astro@latest -- --template blog
```
